# Blynk thermostat

esp8266 thermostat with BME280 sensor
controllable via Blynk.io

Made for a school exercise

## Demo video

[Video demo](https://youtu.be/VmOmwD0J7sU)

## Installation

### Hardware

* ESP8266
* BME280
* Rotary encoder
* I2C oled display
* LED
